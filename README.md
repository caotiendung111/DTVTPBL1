## 🚀 Getting Started

### Prerequisites

- **C++ Compiler** (g++, clang, or MSVC)
- **Make** or another build system (optional)
- Git for version control

### Installation

1. Clone the repository:
```bash
git clone https://github.com/caotiendung111/DTVTPBL1.git
cd DTVTPBL1
```

2. Compile the project:
```bash
g++ -std=c++17 -o program main.cpp
# or use your preferred compiler
```

3. Run the program:
```bash
./program
```
